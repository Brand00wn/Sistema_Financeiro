SOLUÇÃO DO SISTEMA E SUA DESCRIÇÃO
-----------------------------------

-Primeiramente foi feita a criação do Banco de Dados via SQL Server de acordo com a imagem 1 presente no arquivo da prova.

-Em seguida, foi criado o projeto em MVC do sistema no Visual Studio, como pede no PDF.

-Foi gerado um Modelo de Dados utilizando o ADO.NET Entity Data Model (Code First From Database), responsável por gerar as classes da Base de Dados externa.

-Após isto, foi realizada a criação dos Controllers referente às três classes criadas pelo Entity, o que gera também as Views CRUD dessas três.

-CRUD: Para cada classe há 5 Views, Create, Detail, Delete, Edit e Index. O usuário poderá criar, editar, deletar e vizualizar os dados contidos nas classes

-Para o campo "Valor a Pagar" foi criado um método na classe Mensalidade que calcula o Valor da Mensalidade Final. Foram atendidas todas as requisições deste campo relacionadas às alterações do Valor Final, que implicam nos juros ao dia na Mensalidade quando o aluno não realiza o pagamento antes da Data de Vencimento; assim como os descontos das Bolsas de Estudo.
